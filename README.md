## How To Run

1) Run locally
```
npm start
```

2) Build
```
npm run build
```

3) Run Unit Tests
```
npm test
```

4) Run Unit Tests With Code Coverage
```
npm run test:cov
```

5) Run Unit Test Mutation Tests
```
npm run test:mutation
```
